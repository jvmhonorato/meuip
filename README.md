## Getting started

* Dependency
  - Express JS
  - Expo
  - React-native-web
  - React-dom

  
  
* To install the dependencies.
```bash
$ npm install
```
  
  
* To run the project.
```bash
$ expo start
```
  

* Upgrade Expo
```bash
$ expo update <current version> .0.0 
```
